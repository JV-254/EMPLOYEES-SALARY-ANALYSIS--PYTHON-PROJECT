# EMPLOYEES-SALARY-ANALYSIS--PYTHON-PROJECT

## Project Overview

Employee Salary Analysis is a data analysis project built using Python and Pandas to examine employee salary data across different departments. The project provides insights into employee compensation, departmental salary distribution, experience levels, and workforce demographics.

## Features

* Load employee data from a CSV file
* Calculate salary statistics
* Analyze salaries by department
* Identify highest and lowest paid employees
* Rank employees by salary
* Categorize employees by salary level
* Analyze employee experience and age groups
* Filter employees based on custom conditions

## Technologies Used

* Python
* Pandas
* Jupyter Notebook

## Dataset Structure

| Column      | Description                |
| ----------- | -------------------------- |
| Employee_ID | Unique employee identifier |
| Name        | Employee name              |
| Department  | Employee department        |
| Age         | Employee age               |
| Experience  | Years of experience        |
| Salary      | Employee salary            |
